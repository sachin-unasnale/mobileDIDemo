
 This is a simple Project with 3 View Controllers.

 1. Login View Controller
     Login page asks user to enter user name and password.
     username and passwords are hard coded with userName: mobileDI and password: mobileDI123
     after pressing Login Button, these two fields are validated and next View Controller is pushed only if login credentials are submitted correctly. 

2. Task View Controller
     Task View Controller contains one Table View and a Back Button.
     Table View is displayed with custom cells.
     each cell dispalys data received from API.
   
     On Click of each cell, a new View Controller will be pushed on top of it.

3. Task View Controller
     Task View Controller contains one Text View and Back Button.
     Text View is displayed with details of corresponding cell Pressed.
